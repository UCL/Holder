title EIT Software System Overview

EEG System->Data Collector: Send(Raw V data) 
Data Collector->Data Procesor: Send(Averaged V data)
Data Procesor->Data Procesor: Clean and prepare Human data
Data Procesor->Imager: Send(Vbaseline, noise estimation)
Imager->Imager: Reconstruct image
Imager->Visualizer: Send(4D data)
Visualizer->Visualizer: Create(Visualization)
note right of Visualizer: VTK, Matlab, Quantit.
